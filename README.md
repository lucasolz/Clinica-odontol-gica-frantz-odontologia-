# 🦷 Frantz Odontologia - Landing Page

Landing Page moderna, totalmente responsiva e de alta conversão desenvolvida para a clínica Frantz Odontologia, localizada em Butiá - RS.

## 🚀 Demonstração Online
O projeto está publicado e pode ser testado em tempo real através do link: 
👉 [frantz-odontologia.netlify.app](https://pagina-web-odontologia.netlify.app/)

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3 Puro:** Estilização, layout flexível (Flexbox) e responsividade para celular e PC.
- **Animações CSS:** Transições dinâmicas ao interagir (`:hover` e `:active`) nos cards de serviços e nos botões de chamada para o WhatsApp.

## 🎯 Funcionalidades Principais
- Navegação fluida por âncoras internas.
- Links inteligentes de agendamento direto via WhatsApp para cada procedimento.
- Integração externa com a ficha do Google Maps da clínica.
